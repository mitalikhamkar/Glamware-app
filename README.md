# Glamware 👗

Glamware is a virtual wardrobe and outfit styling mobile application.

## 🚀 Features
- Avatar creation
- Upload clothing items
- Try outfits digitally (one item per category)
- Save outfits

## 🛠️ Tech Stack
- React Native
- Node.js
- Express.js
- MongoDB

## 📂 Project Structure
- Backend: https://github.com/mitalikhamkar/glamware-backend
- 3D Assets: https://github.com/mitalikhamkar/glamware-assets

## 💡 Key Learnings
- Debugging 3D rendering issues
- User-specific data handling
- API integration
